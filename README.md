# Fishing-clicker-game
## Overview
This is a simple, 2D, pixel art video game where you click buttons to fish. You can catch different species and store them in an inventory to sell for cash. The game is still in development, so you cannot run it yet.

## Features
- You can fish for fish
  - The name of the fish you catch will show up in the text above the button
- You can open an inventory and click icons to sell fish for money
  - The icons in the inventory are placeholders, and the color corresponds to the rarity of the fish.
  - Green - Common | Blue - Rare | Red - Epic | Gold - Mythical
  - They each cost a different amount when you sell them.
  
## Future steps
- Add real icons for fish in inventory
- fix inventory bugs (selling and icon spawn placement)
- Polish the game and make it more aesthetic
- Add more features, such as more fish species and a shop with upgrades
